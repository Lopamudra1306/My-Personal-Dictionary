# 📖 My Personal Dictionary

A modern and responsive dictionary web application built using **HTML, CSS, and JavaScript**. It allows users to search for English words and instantly view their meanings, phonetics, pronunciation, and usage examples through the Free Dictionary API. Users can also save favorite words, switch between light and dark themes, and download word details as PDF files.

## ✨ Features

- 🔍 Search any English word instantly
- 📚 View word definitions and examples
- 🔊 Listen to pronunciation with audio
- 📝 Display phonetic transcription
- ⭐ Save favorite words using Local Storage
- 🌙 Toggle between Light and Dark Mode
- 📄 Download word details as a PDF
- ⌨️ Press **Enter** to search quickly
- 📱 Fully responsive and mobile-friendly design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- DictionaryAPI.dev
- jsPDF

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/my-personal-dictionary.git
   ```

2. Open the project folder.

3. Launch `index.html` in your browser.

No installation or build process is required.
